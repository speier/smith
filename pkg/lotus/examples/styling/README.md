# Styling Example

Demonstrates text styling and layout features.

## Run it

```bash
go run main.go
```

## Features shown

- Text styles: bold, italic, underline, strikethrough, dim, reverse
- Colors: hex colors (`#ff0000`) and named colors (`bright-cyan`)
- Text alignment: left, center, right
- Text overflow: ellipsis
- Borders: single, rounded, double
- Border colors
- Max lines with ellipsis
- Visibility control

All styles use CSS-like API for familiar developer experience.

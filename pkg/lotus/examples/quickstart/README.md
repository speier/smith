# Quickstart

Your first Lotus app - demonstrates the minimal setup.

## Run it

```bash
go run main.go
```

## What you'll learn

- Functional components with `lotus.Run(func(ctx) { ... })`
- Declarative UI with `VStack`, `Text`
- Text styling with `.WithBold()`, `.WithColor()`
